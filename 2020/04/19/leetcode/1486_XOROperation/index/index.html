1486.数组异或操作

题解

前言

统计 [2,n]\tt{[2, n]}[2,n] 中质数的数量是一个很常见的题目，也有很多巧妙高效的做法，接下来的部分只会讲述一些常见的做法，更多的拓展内容读者可以自行搜索补充，也欢迎在评论区与大家分享交流。

方法一：模拟

思路：

按照题意模拟即可：

  • 初始化 ans=0\tt{ans = 0}ans=0
  • 遍历区间 [0,n−1]\tt{[0, n - 1]}[0,n−1] 中的每一个整数 i\tt{i}i，令 ans\tt{ans}ans 与每一个 start+2×i\tt{start + 2 \times i}start+2×i 做异或运算
  • 最终返回 ans\tt{ans}ans，即我们需要的答案

复杂度分析：

  • 时间复杂度：O(n)\rm{O(n)}O(n)。这里用一重循环对 n\tt{n}n 个数字进行异或。
  • 空间复杂度：O(1)\rm{O(1)}O(1)。这里只是用了常量级别的辅助空间。

方法二：数学

记 ⊕\tt{\oplus}⊕ 为异或操作，异或操作满足以下性质：

  1. x⊕x=0\tt{x \oplus x = 0}x⊕x=0
  2. x⊕y=y⊕x\tt{x \oplus y = y \oplus x}x⊕y=y⊕x（交换律）
  3. (x⊕y)⊕z=x⊕(y⊕z)\tt{(x \oplus y) \oplus z = x \oplus (y \oplus z)}(x⊕y)⊕z=x⊕(y⊕z)（结合律）
  4. x⊕y⊕y=x\tt{x \oplus y \oplus y = x}x⊕y⊕y=x（自反性）
  5. ∀i∈Z\tt{\forall i \in Z }∀i∈Z，有4i⊕(4i+1)⊕(4i+2)⊕(4i+3)=0\tt{4i \oplus (4i + 1) \oplus (4i + 2) \oplus (4i + 3) = 0}4i⊕(4i+1)⊕(4i+2)⊕(4i+3)=0

在本题中，我们要计算

start⊕(start+2i)⊕(start+4i)⊕⋯⊕(start+2(n−1))\tt{start \oplus (start + 2i) \oplus (start + 4i) \oplus \cdots \oplus (start + 2(n - 1)) } start⊕(start+2i)⊕(start+4i)⊕⋯⊕(start+2(n−1))

观察公式可以知道，这些数的奇偶性相同，因此它们的二进制表示中的最低位均为 1\tt{1}1，或者均为 0\tt{0}0。于是我们可以把参与运算的数的二进制位的最低位提取出来单独处理，当且仅当 start\tt{start}start 为奇数，且 n\tt{n}n 也为奇数时，结果的二进制位的最低位才为 1\tt{1}1。

此时我们可以将公式转化为：

(s⊕(s+1)⊕(s+2)⊕⋯⊕(s+n−1))×+ e\tt{(s \oplus (s + 1) \oplus (s + 2) \oplus \cdots \oplus (s + n - 1)) \times + \ e} (s⊕(s+1)⊕(s+2)⊕⋯⊕(s+n−1))×+ e

其中 s=⌊start2⌋\tt{s = \lfloor \frac{start}{2} \rfloor}s=⌊2start​⌋，e\tt{e}e 表示运算结果的最低位。即我们单独处理最低位，而舍去最低位后的数列恰成为一串连续的整数。

这样我们可以描述一个函数 sumXor(x)\tt{sumXor(x)}sumXor(x)，表示 0⊕1⊕2⊕⋯⊕x\tt{0 \oplus 1 \oplus 2 \oplus \cdots \oplus x}0⊕1⊕2⊕⋯⊕x。利用异或运算的性质5，我们可以将计算该函数的复杂度降低到 O(1)\rm{O(1)}O(1)，因为以 4i\tt{4i}4i 为开头的连续四个整数异或的结果为 0\tt{0}0，所以 sumXor(x)\tt{sumXor(x)}sumXor(x) 可以被表示为：

sumXor(x)={x,x=4k,k∈Z(x−1)⊕x,x=4k+1,k∈Z(x−2)⊕(x−1)⊕x,x=4k+2,k∈Z(x−3)⊕(x−2)⊕(x−1)⊕x,x=4k+3,k∈Z\tt{sumXor(x) = \begin{cases} \tt x, && \tt x=4k, k \in Z \\\\ \tt (x-1) \oplus x, && \tt x =4k+1, k \in Z \\\\ \tt (x-2) \oplus (x-1) \oplus x, && \tt x=4k+2,k \in Z \\\\ \tt (x-3) \oplus (x-2) \oplus (x-1) \oplus x, && \tt x=4k+3,k \in Z \end{cases} } sumXor(x)=⎩⎨⎧​x,(x−1)⊕x,(x−2)⊕(x−1)⊕x,(x−3)⊕(x−2)⊕(x−1)⊕x,​​x=4k,k∈Zx=4k+1,k∈Zx=4k+2,k∈Zx=4k+3,k∈Z​

我么可以进一步化简改式：

sumXor(x)={x,x=4k,k∈Z1,x=4k+1,k∈Zx+1,x=4k+2,k∈Z0,x=4k+3,k∈Z\tt{sumXor(x) = \begin{cases} \tt x, && \tt x=4k, k \in Z \\\\ \tt 1, && \tt x =4k+1, k \in Z \\\\ \tt x+1, && \tt x=4k+2,k \in Z \\\\ \tt 0, && \tt x=4k+3,k \in Z \end{cases} } sumXor(x)=⎩⎨⎧​x,1,x+1,0,​​x=4k,k∈Zx=4k+1,k∈Zx=4k+2,k∈Zx=4k+3,k∈Z​

这样最后的结果即可表示为 (sumXor(s−1)⊕sumXor(s+n−1))×2+e\tt{(sumXor(s-1) \oplus sumXor(s+n-1)) \times2 + e}(sumXor(s−1)⊕sumXor(s+n−1))×2+e

复杂度分析：

  • 时间复杂度：O(1)\rm{O(1)}O(1)。我们只需要常数的时间计算出结果。
  • 空间复杂度：O(1)\rm{O(1)}O(1)。我们只需要常数的空间保存若干变量。
